About Our App!


Happier Hour - A Narrative

We are in a rigorous MS program at the University of Texas at Austin. The MS in IT & Management program in the McCombs School of Business is heavily interested in the full development of their students to begin successful careers. We achieve this by emphasizing the business context without losing focus on the technical knowledge necessary to contribute in the tech industry.

So begin the 2019 Summer Semester. Here's what we know about the Advanced Programming and Application Development course:

  • The semester is 6 weeks long
  • Learn Several Concepts
    1. Python Programming (for students who have not been exposed)
    2. Git version control for paired programming project
    3. Database Concepts
    4. Client-Server Architecture
    5. Security, Authorization, Permissions
    6. Web Application Development
    7. Deployment on Google Cloud Platform
    8. API Concepts
    9. Android App Development
  • Determine an Application to Develop
  • Create a Web App using Python's Django framework
    1. Expose API's on this Web App
    2. Deploy it on Google App Engine, such that it can be accessed via happierhour.appspot.com
    3. Support file management system and implement connection with Google Storage
  • Develop an Android app in Kotlin
    1. Connect it to the API's
    2. Utilize Token Authentication
    3. Support Main Functionality of the WebApp in a Mobile Native App
    4. Deploy the APK to be downloaded by the public Click Here to Download
  • Finish on time!

But We Care About Our App!

Our idea was too great. Yelp is too restaurant friendly and old. Google Maps is solving too many problems in one app and cannot tailor the experience extremely well for a niche purpose. We are young professionals who care about making sure we find the best drinks and the best food deals at Happy Hours near us! We want a better way to search for, and filter through bars and restaurants that have Happy Hour deals.

Our platform is the key to having the Happiest Happy Hours

You can access our Github Repository or visit our web application. And if you have an Android Phone, please download our APK file to test out our functionality.
